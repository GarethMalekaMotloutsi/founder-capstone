## 📖 Overview

Founder Capstone is a modern marketplace application developed as part of a software development capstone project. The goal was to build a clean, responsive, and user-friendly marketplace experience while applying modern front-end development principles and best practices.

The project focuses on creating reusable React components, maintaining strong TypeScript typing, building responsive layouts, and delivering an intuitive user experience.

---

## ✨ Features

- Responsive user interface
- Product browsing experience
- Detailed product pages
- Reusable React components
- TypeScript type safety
- Clean component architecture
- Mobile-friendly layout
- Modern UI styling
- Fast performance with Vite

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/GarethMalekaMotloutsi/founder-capstone.git
```

Navigate into the project

```bash
cd founder-capstone
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
